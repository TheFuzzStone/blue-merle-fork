# US first names for SSID masquerading.
#
# Used to compose SSIDs of the form "<Name>'s iPhone" — the exact
# pattern iOS uses for its Personal Hotspot broadcast.
#
# Rules:
#   * one name per line, ASCII letters only (guard against uci escaping
#     issues and non-Latin surprises);
#   * lines starting with '#' and blank lines are ignored;
#   * you can freely edit this file after install:
#       vi /lib/blue-merle/us-first-names.txt
#     changes take effect on the next boot (or `service blue-merle reload`).
#
# The list mixes male, female and gender-neutral names, roughly
# reflecting Social Security Administration top names for people
# currently 18-45 in the US (a plausible smartphone-owner demographic
# in 2025). ~250 entries, so a random pick has a small chance of
# repeating between rotations.
