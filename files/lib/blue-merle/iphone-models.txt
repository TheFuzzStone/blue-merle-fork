# Realistic iPhone model names for hostname masquerading.
#
# Rules:
#   * one model per line, no spaces (dashes only) so the value is a
#     RFC 952 / 1123 valid hostname;
#   * lines starting with '#' and blank lines are ignored;
#   * the list intentionally starts at iPhone X (2017) — older models
#     look conspicuously out of place on a public WiFi in 2025+ and
#     would be more, not less, of a fingerprint;
#   * you can freely edit this file after install:
#       vi /lib/blue-merle/iphone-models.txt
#     changes take effect on the next boot (or after
#     `service blue-merle reload`).
